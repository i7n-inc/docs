# Memories

ATX persists learnings — resolved issues, reviewer feedback, project
conventions — in SQLite so every subsequent review is better informed.

## Inspect

```bash
atx project memory list
atx project memory recommendations
```

`recommendations` lists agent-update recommendations produced from
accumulated learnings.
