# Dashboards

Once the daemon is running, the dashboard is at
[`http://localhost:30000/dashboard`](http://localhost:30000/dashboard).

## What's on it

- **Projects List** — every registered project with status, agent count,
  and session count.
- **Project Detail** — click a project to see its agents and sessions.
- **Session Detail** — full conversation logs and Mermaid sequence
  diagrams for each review session.
