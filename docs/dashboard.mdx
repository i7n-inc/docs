# Dashboard

Once the daemon is running, the dashboard is at
[`http://localhost:30000/dashboard`](http://localhost:30000/dashboard).

![ATX dashboard landing page](/img/screenshots/dashboard-landing.png)

## What's on it

- **Dashboard (landing)** — cost/successful task (median), time/successful
  task (median), cost by model, and 8-week trends for memory and task
  throughput.
- **Projects** — every registered project with status, agent count, and
  session count. Drill in for agents and sessions per project.
- **Reviews** — cross-project reviews table sorted by most recent.
- **Memories** — approved + applied learnings across projects.
- **Models** — catalog by provider (what ATX will pick from).
- **Session detail** — full conversation logs and Mermaid sequence
  diagrams for each review.
